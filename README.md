# coachway-tilbud-data

Weekly snapshot of public grocery offers (Netto, REMA 1000, Lidl) from the public Tjek API, fetched every Monday 05:00 UTC by GitHub Actions.

Consumed by the Coachway recipe routine via `https://raw.githubusercontent.com/Markus-Evers/coachway-tilbud-data/main/offers.json`.

Data is public discount-flyer information. No credentials involved.
